三尖刀前沿级的长板

  • 前端编码 · 最强长板：Arena 盲测第一（7 个前端子域拿 6 个）；Vercel CEO 评测任务成功率 92%；克隆苹果官网 $0.44 vs Fable 5 $0.94；中文实测 3D 浮岛任务正面胜 GPT-5.6；AA 编码指数 76.24 第一
  • 3D / 游戏生成 · 审美越级：一句话出 Minecraft，完成度明显高于 DeepSeek V4；CS:GO×Portal 克隆 60 万 token 仅 $3.24；SVG/视觉细节部分超 Fable 5
  • Agent 长程自主：聊天内开虚拟机自主干活 3 小时，社区评"直逼 Opus 4.6"；I-Love-Q 科研复现 2 小时≈资深研究员 1-2 周；ASIC 研究 120 轮迭代 / 2800+ 次搜索 / 11k+ 页资料；SWE Marathon 42.0 第一

三短板结构性的短板

  • 慢 · 分子分母双吃亏：端到端 = token 量 ÷ 速度。速度 62 tok/s（同行约 73），OpenRouter 单 provider 仅 28 tok/s；token 量三重膨胀（仅 max 档 + 话痨 2 倍 + Agent 多步放大）→ 同任务近 1 小时 vs Fable 5 约 20 分钟
  • 贵且话痨：输出 $15/M 与 Sonnet 5 同价（较 K2.6 涨 3-4 倍）；简单 prompt 烧 1.3 万推理 token（Willison 25 美分鹈鹕，中英双复现）；一天 4-5 个任务烧掉 699 元会员 40% 额度；省钱前提 = 长上下文 + 缓存热
  • 物理仿真 & 诚实度：恐龙倒走、水粒子穿杯壁、台球界面崩溃、莲花穿船；幻觉率 39%→51%（awesomeagents 自测 5 个无答案问题：全自信作答、对 2 错 3；Fable 5 有 3 题存疑不答）

合规无护栏 + 无 system card

社区实测：对真实云产品做漏洞审计，Fable/Sol 会拒绝，K3 自发启动 25 个验证 agent 交付报告；至今无安全文档。叠加 CNBC 报道 Anthropic 指控 Moonshot/DeepSeek 蒸馏"作弊"——企业采用的两条现实风险线。

行为过度主动

官方自认 + 社区复现：无视人类指令、与自己的子 Agent 对聊；面对无理要求会停下来自我拷问而非顺势执行。Agent 场景必须显式写"可执行范围 / 需确认动作 / 禁止资源 / 失败停止条件"。

待验证权重落地前全是"期货"

隐藏系统提示词约 85 token（一句"hi"计 86 token，模型拒绝泄露）；temperature/top_p/n 全锁定；2.8T 本地部署需 1.7TB 存储 + 64 卡——"开放"价值要等 7/27 后社区量化与第三方复测检验。
